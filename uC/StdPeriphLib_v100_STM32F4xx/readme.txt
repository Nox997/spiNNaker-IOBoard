StdPeriphLib_v100_STM32F4xx
===========================

This library project contains a port of the STM32F4xx Standard 
Peripherals Library (v.1.0.0, 30-September-2011) suitable for
use within Red Suite.

For general details of the STM32F4xx Standard Peripherals Library,
including license and pointers to STM's documention, please see
the files within the 'stm_docs' subdirectory of this project.

Note that this project requires the CMSISv2p10_STM32F4xx library 
project to exist in the same workspace.
